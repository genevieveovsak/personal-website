# genevieveovsak.com

Personal website for Genevieve Ovsak — product strategist, systems thinker, and human-first technologist.

## Stack

- [Astro](https://astro.build/) — static site generator
- Vanilla CSS with custom properties — no CSS framework
- Fonts via [@fontsource](https://fontsource.org/) — self-hosted, no CDN dependency
  - Ephesis (hero name)
  - Lora (headings, display)
  - DM Sans (body)

## Development

```bash
npm install
npm run dev
```

Runs at `http://localhost:4321`.

## Build

```bash
npm run build
```

Output goes to `dist/`.
